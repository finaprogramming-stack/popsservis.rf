Настройка бесшовных карточек.

После добавления фото запускай:
node scripts/generate-gallery.js

Генератор сам обновит JSON каждого раздела.
Внутри будет только:

[
  { "name": "1", "count": 0 }
]

name - имя фото без расширения.
count - сколько фото склеить в одну карточку, начиная с этого фото.

Пример:
[
  { "name": "1", "count": 3 },
  { "name": "4", "count": 0 }
]

Это значит: фото 1, 2, 3 будут одной бесшовной карточкой.
0 значит обычная одиночная карточка.

Папки разделов:

bitumovoz - Битумовоз
vodnaya-tehnika - Водная техника
drugoe - Другое
kmd - КМД
krany-kmu - Краны и КМУ
kuzova-ramy - Кузова и рамы
svar-raboty - Свар. работы
traly-pricepy - Тралы и прицепы
truby-metallokonstrukcii - Трубы и металлоконструкции
fasady - Фасады

Описание всего раздела редактируется тут:
gallery-sections/descriptions.json
